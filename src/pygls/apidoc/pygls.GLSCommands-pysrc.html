 1  """ 
 2  Python library for GPS Location Sharing - collection of constants for commands and replies for the GLS server. 
 3   
 4  All constants starting with "co" are commands sent from the client to the server. All constants starting with 
 5  "re" are replies, which are sent back from the server to the client in reply to a request. 
 6   
 7  http://www.assembla.com/wiki/show/dZdDzazrmr3k7AabIlDkbG 
 8   
 9  @author: Michael Pilgermann 
10  @contact: mailto:michael.pilgermann@gmx.de 
11  @contact: http://www.kichkasch.de 
12  @license: GPL (General Public License) 
13   
14  @var CO_VERSION: Command string of protocol - request version. 
15  @type CO_VERSION: C{String} 
16  @var CO_LOGIN: Command string of protocol - login attempt. 
17  @type CO_LOGIN: C{String} 
18  @var CO_DEVICE: Command string of protocol - submit device information. 
19  @type CO_DEVICE: C{String} 
20  @var CO_POSITION: Command string of protocol - submit or request position information. 
21  @type CO_POSITION: C{String} 
22  @var CO_WAYPOINT: Command string of protocol - submit or request waypoint information. 
23  @type CO_WAYPOINT: C{String} 
24  @var CO_GROUP: Command string of protocol - enter group or request group information. 
25  @type CO_GROUP: C{String} 
26  @var CO_QUIT: Command string of protocol - shut down connection. 
27  @type CO_QUIT: C{String} 
28   
29  @var RE_OK: Command reply from server of protocol - OK, sucessful processed last request. 
30  @type RE_OK: C{String} 
31  @var RE_CHANGE: Command reply from server of protocol - processing not sucessful because of business rules. 
32  @type RE_CHANGE: C{String} 
33  @var RE_ERROR: Command reply from server of protocol - error, due to validation rules. 
34  @type RE_ERROR: C{String} 
35  @var RE_GROUP: Command reply from server of protocol - sending of a group. 
36  @type RE_GROUP: C{String} 
37  @var RE_POSITION: Command reply from server of protocol - sending position of a client. 
38  @type RE_POSITION: C{String} 
39  @var RE_WAYPOINT: Command reply from server of protocol - sending a waypoint of a client. 
40  @type RE_WAYPOINT: C{String} 
41  @var RE_FINISHED: Command reply from server of protocol - a sequence was finished. 
42  @type RE_FINISHED: C{String} 
43  @var RE_QUIT: Command reply from server of protocol - server is closing connection. 
44  @type RE_QUIT: C{String} 
45  """
