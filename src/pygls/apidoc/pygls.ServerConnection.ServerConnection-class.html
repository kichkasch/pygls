Class ServerConnection

source code

An instance of this class maintains one connection to the GLS server.

It may be reused for several commands.

Connection will be established automatically as soon as the first command is to be sent to the server. Connection will be shut down from the client as soon as the destructor is called.

_sendCommand(self, command, initMode=0)

source code 

Sends exactly one command to the GLS server.

The command is extended by a line feed before sending. The line feed from the line(s) in the reply is / are removed before the value is returned.

Replies, which are made up by several lines are returned as a list of Strings (in contrast to single line replies, which are returned as a single String). Multi-Line-Replies are currently indicated by a leading "P" (position of others) ,a leading "W" (waypoints of others) or a leading "G" (available groups). The sequence is terminated by a line made up by an "F". This last line will be part of the returned list as well.

The replies are checked for error messages. If the reply is either a "C" or an "E" an GLSException.GLSException will be raised.
Parameters:
  • command (String) - Command to be sent to the server
  • initMode (int) - When sending this command the connection is in initialisation mode; meaning, it shall no be checked, whether a connection is establised already
Returns: String or List of String
Received reply from the server. A String in case of a single line reply; an array of strings in case of multi line reply.

_sendCommands(self, commands, initMode=0)

source code 

Sends an array of commands to the GLS server (line by line).

Each item in the lists is sent to the GLS server. Before sending, a line feed is appended.

Received results are stored in a dictionary - the list of commands makes up the keys in the dictionary, the received results are the values. (Values might either be a simple string or a list of strings in case of several lines are sent back by the server for a command.
Parameters:
  • commands (List) - Commands to be send to the GLS server. They must not contain the line feed.
  • initMode (int) - When sending this command the connection is in initialisation mode; meaning, it shall no be checked, whether a connection is establised already
Returns: Dict of String and String | List
Received results for all command

_establishConnection(self)

source code 

Starts up the connection to the GLS server.

The method is going through the full handshake (V, N, D) with the GLS server. If a password is available for this connection, it is used - if no password is available, the connection is being established without providing one.

The socket is stored in the instance reference (_s) and the state of the connection (_connected) is set to 1. The reply of the server with the supported versions is stored in the instance reference (_serverVersion).
